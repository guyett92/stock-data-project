The above application pulls data from ClearBit's API to serve logos. The stock data comes from Alpha Vantage's API and the name from the stock symbol comes from World Trading Data's API. The application parses the API data to serve it's best guess at a logo. If the logo is wrong, there is a form to fill out in order to help me improve the way the code identifies a match. The application was created because a simple and clean UI did not exist for stock data that is accompanied by a logo.
